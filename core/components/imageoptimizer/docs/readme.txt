ImageOptimizer для MODX Revolution 3.x

Конвертация изображений в WebP/AVIF, responsive srcset по breakpoints, очередь задач
и автоматическая инъекция <picture> в HTML без правки чанков. Vue-админка (VueTools).

Полная документация: https://docs.modx.pro/components/imageoptimizer/
